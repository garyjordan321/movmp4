What is Scar AI?

Scar AI is a built-in Chrome extension that's compatible with DexScreener, Photon, and BullX (more coming soon). It shows users key bundle metrics about a token as well as the token creator's data.

What’s the cost?

The extension is completely free for all users. We aim to provide transparent and accessible tools for all traders. In future we look to provide a premium feature for our token holders.

How to install the extension?

You can download the extension from the Chrome Web Store or directly from our official website. Installation takes only a few clicks!
